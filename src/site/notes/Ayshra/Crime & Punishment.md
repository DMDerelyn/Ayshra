# Crime & Punishment:

**Misdemeanors**:
- D - 15g
- C -  20g
- B - 30g
- A - 60g

**Felonies**:
- E - 150g
- D - 250g
- C - 350g
- B - Prison Only
- A - Prison Only

If a person has gained money or property through the commission of the offense, the fine is doubled.
